# Конфигурация приложения

## Параметры

| Параметр         | Тип     | Описание                         | Значение по умолчанию  |
|------------------|---------|----------------------------------|------------------------|
| `server.host`    | string  | IP-адрес, на котором слушает API | `0.0.0.0`              |
| `server.port`    | string  | Порт сервера                     | `8080`                 |
